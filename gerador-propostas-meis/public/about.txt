# Gerador de Propostas MEI

## Sobre a Aplicação
Aplicação web moderna para criação de propostas comerciais profissionais especificamente desenvolvida para Microempreendedores Individuais (MEI). 

**URL de produção:** https://gerador-propostas-meis.vercel.app

## Funcionalidades Principais
- Criação de propostas comerciais personalizadas
- Upload de logotipo empresarial
- Personalização de cores
- Geração de PDF profissional
- Compartilhamento via WhatsApp
- Interface totalmente responsiva

## Tecnologias
- React 18
- Vite
- Tailwind CSS
- Radix UI
- jsPDF
- Vercel Analytics
